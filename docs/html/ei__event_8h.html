typedef ei_bool_t(* ei_default_handle_func_t) (struct ei_event_t *event)

A function that is called in response to an event that has not been processed by any widget.

Parameters
eventThe event containing all its parameters (type, etc.)
Returns
EI_TRUE if the function handled the event, EI_FALSE otherwise, in this case the event is dismissed.
